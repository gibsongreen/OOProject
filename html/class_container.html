void Container::saveContainerFile ( )

Save the inventory of the food and drink list to a text file.

This function allows the user to save the inventory of the container, both the food list and drink list, to a text file for the container. The container file can then be read and edited by a returning user. This function should only be called after editing the contents of the inventory of the container.
