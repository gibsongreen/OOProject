Change the internal temperature of the Freezer container.

This function allows the user to change the internal temperature of the freezer instance. The freezer inherits changeInternalTemperature() from the Refridgerator class, the freezer temperature will print and the user will be prompted to enter a new internal temperature for the freezer to be set to, within a predetermined range.
